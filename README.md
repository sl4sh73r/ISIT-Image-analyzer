# ISIT-Image-analyzer

Простое приложение для анализа изображений и сравнения результатов двух LLM-vision моделей (qwen3-vl-4b и gemma-3-4b) через LM Studio.

Ключевые моменты:
- Интерфейс загрузки изображений и последовательный анализ моделей
- Поддержка LM Studio REST API (рекомендуется) и попытки SDK (в ветке `app_sdk.py`)
- Показ метрик: время обработки, токены, скорость

Быстрый старт

```bash
# активировать виртуальное окружение
source .venv/bin/activate

# установить зависимости
pip install -r requirements.txt

# запустить приложение
python app.py

# открыть в браузере
http://127.0.0.1:5001
```

Примечания

- LM Studio: убедитесь, что REST API Server запущен (порт 1234) и необходимые модели загружены в LM Studio (My Models).
- `app_sdk.py` содержит экспериментальную реализацию с Python SDK (может не работать в текущей версии LM Studio).
